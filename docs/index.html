About the Project

meta is your go-to tool for performing comprehensive meta-analysis across diverse fields. Harness the power of data integration to uncover hidden insights and trends that individual studies might miss.

Getting Started

  1. Clone the repository: git clone https://github.com/abikahs/meta.git
  2. Navigate to the project directory: cd meta/main
  3. Install dependencies: pip install -r requirements.txt
